
msdgames - Bootstrap 4 Light Gaming Download Template
---------------------------------------------------

How to use:
1. Download or clone this folder to your computer.
2. Create a GitHub repository named <username>.github.io (or any repo and enable GitHub Pages).
3. Upload the contents of this folder to the repository root (index.html at root).
4. Optionally edit files: assets/css/style.css and the HTML pages.
5. Replace placeholder images in assets/images with real game posters.
6. Be sure to update privacy.html with your real policy before publishing.
